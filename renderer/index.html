欢迎使用 OpenClaw Desktop Client

开始使用：

  1. 点击右上角的 ⚙️ 设置按钮
  2. 在设置中选择 Config
  3. 配置你的 Gateway URL 和 token
  4. 点击 Save & Connect
  5. 在左侧选择一个会话或点击 + 新建对话 创建新会话
  6. 开始聊天！
